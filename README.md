# PORT_SCAN
This is a port scanner that detects the open or closed ports of a network. 

INSTALLATION:

    git clone https://github.com/BO3K404/PORT_SCAN.git
    cd PORT_SCAN
    bash portscan.sh

EXECUTION:

python portscan.py -H -P
EXAMPLE:

python portscan.py -H 192.168.1.254 -P 902
